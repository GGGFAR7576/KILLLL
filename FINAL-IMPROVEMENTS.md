## 🚀 Как использовать

### Запуск:
```bash
cd viber-breach-system
npm start
```

Или двойной клик на:
```
START-APP.bat
```

### Перетаскивание модального окна:
1. Нажми кнопку "ADVANCED CONFIG"
2. Зажми левую кнопку мыши на заголовке
3. Двигай окно куда хочешь
4. Отпусти мышь
